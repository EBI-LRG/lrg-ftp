This private ftp site contains files relevant to the LRG project and is a
resource for file exchange between EBI and NCBI.

This directory contains the following directories:

    ./NCBI      -   Files created or modified by NCBI
    ./EBI       -   Files created or modified by EBI
    ./Pending   -   Files pending final approval before release
    ./Published -   Files that have been released to the public
    
The contents of the 'Pending' and 'Published' directories should be the same as
what is available on the public ftp site: 
ftp://ftp.ebi.ac.uk/pub/databases/lrgex/
