This directory contains the following files:

- LRG.rnc - LRG XML RelaxNG schema (v1.6)
- README.txt - readme for schema
- relnotes.txt - Release notes
- lrg2html.xsl - XSLT document for formatting LRG XML as HTML
- lrg2txt.xsl - XSLT document for formatting LRG XML as text
- wordwrap.xml - XML document required for formatted HTML (must be in same directory)
- LRG_1.xml - LRG XML file for COL1A1
- LRG_2.xml - LRG XML file for COL1A2
- LRG_3.xml - LRG XML file for COL1A3
- LRG_4.xml - LRG XML file for CRTAP
- LRG_6.xml - LRG XML file for ATP1A2
- LRG_7.xml - LRG XML file for CACNA1A
- LRG_8.xml - LRG XML file for SCN1A
- LRG_10.xml - LRG XML file for PPIB
- LRG_12.xml - LRG XML file for FKBP10
- LRG_13.xml - LRG XML file for CALCA
- LRG_15.xml - LRG XML file for UBE3A
